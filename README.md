Mesto Russia
-------------

Сервис Mesto Russia позволяет добавлять, удалять фотокарточки, ставить лайки и изменять данные профиля на странице.

В данной проектной работе использовались:

- Методология БЭМ для CSS.
- Методология ООП для JS.
- Адаптивная верстка.
- JS.
